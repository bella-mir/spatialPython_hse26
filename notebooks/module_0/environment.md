# Настройка рабочей среды

В этом разделе мы рассмотрим, как настроить рабочую среду: какой редактор кода выбрать, как создать виртуальное окружение и как организовать хранение рабочих файлов.

> **Рекомендуемая версия Python: 3.11.14.** Именно с ней тестировались все материалы курса, установка и совместимость библиотек.

## Инструменты

В курсе мы работаем в **Visual Studio Code (VS Code)** с расширением **Jupyter**. Это позволяет запускать `.ipynb`-ноутбуки прямо в редакторе — без запуска отдельного браузерного приложения.

**Jupyter Notebook** — интерактивная среда для создания документов, содержащих код, текст, изображения и результаты вычислений. Код выполняется в отдельных ячейках, что делает работу с данными удобной и наглядной.

**Visual Studio Code (VS Code)** — лёгкий и мощный редактор кода от Microsoft, который через расширение Jupyter поддерживает работу с ноутбуками.

## Что такое ядро (Kernel)

Прежде чем приступить к установке, полезно понять, как работает Jupyter.

**Ядро (kernel)** — это фоновый процесс, который выполняет код из ноутбука. Когда вы нажимаете `Shift + Enter`, код из ячейки отправляется в ядро, ядро его выполняет и возвращает результат обратно в ноутбук.

Ядро привязано к конкретному виртуальному окружению Python. Поэтому важно, чтобы при открытии ноутбука было выбрано правильное окружение — иначе нужные библиотеки могут быть недоступны.

Если ядро не работает или зависло — код выполняться не будет. В таком случае его можно перезапустить: `Ctrl+Shift+P → Jupyter: Restart Kernel`.

### 1. Установка необходимых инструментов

#### 1.1. Установите VS Code

1. Скачайте с официального сайта: [https://code.visualstudio.com](https://code.visualstudio.com).
2. Запустите установщик и следуйте инструкциям.

#### 1.2. Установите uv

`uv` — современный и очень быстрый менеджер пакетов и виртуальных окружений для Python. Он умеет самостоятельно скачивать и устанавливать нужные версии Python, создавать изолированные окружения и управлять зависимостями.

Можете также изучить другие варианты установки в документации [uv](https://docs.astral.sh/uv/getting-started/installation/)

**macOS / Linux:**

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

**Windows (PowerShell):**

```powershell
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

После установки перезапустите терминал и проверьте:

```bash
uv --version
```

#### 1.3. Установите Python 3.11.14 через uv

Вместо того чтобы скачивать Python вручную с python.org, используем `uv` — он сам загрузит нужную версию:

```bash
uv python install 3.11.14
```

Проверьте, что версия доступна:

```bash
uv python list
```

#### 1.4. Установите расширения Jupyter и Python в VS Code

1. Откройте VS Code.
2. Перейдите во вкладку **Extensions** (`Ctrl+Shift+X`).
3. Найдите **Jupyter** (разработчик: Microsoft) и установите.
4. Найдите **Python** (разработчик: Microsoft) и установите.

### 2. Настройка рабочей директории и создание нового ноутбука

Рекомендуем выделить одну папку для всех проектов и упражнений курса.

#### 2.1. Создайте пустую папку в любом месте на компьютере.

#### 2.2. Откройте её в VS Code

**File → Open Folder** или перетащите папку в окно VS Code.

#### 2.3. Создайте Jupyter Notebook

В панели **Explorer** нажмите иконку **New File** и введите название файла с расширением `.ipynb`.

#### 2.4. Откройте созданный файл.

### 3. Виртуальное окружение с uv

`uv` создаёт окружение быстро и сразу с нужной версией Python.

#### 3.1. Инициализируйте проект

Откройте терминал в VS Code (**Terminal → New Terminal**) и выполните:

```bash
uv init
```

Команда создаст файлы `pyproject.toml` и `.python-version` — они описывают проект и фиксируют версию Python. Также будет автоматически создана папка `.venv` с виртуальным окружением.

#### 3.2. Проверьте версию Python в окружении

```bash
uv run python --version
```

Если версия не `3.11.14`, задайте её явно:

```bash
uv python pin 3.11.14
```

После этого пересоздайте окружение:

```bash
uv sync
```

#### 3.3. Активируйте окружение вручную (при необходимости)

```bash
# Linux / macOS
source .venv/bin/activate

# Windows
.venv\Scripts\activate
```

> После активации VS Code должен автоматически подхватить окружение для ноутбука. Если нет — выберите его вручную через **Select Kernel** в правом верхнем углу редактора.

### 4. Установка библиотек

#### Рекомендуемый способ — через uv

В репозитории нашего учебника есть файл `requirements.txt` со всеми необходимыми библиотеками. Скачаетй его, добавьте в рабочую директорию и выполните следующую команду:

```bash
uv add -r requirements.txt
```

`uv` установит пакеты, обновит `pyproject.toml` и создаст `uv.lock` — файл, фиксирующий точные версии зависимостей.

#### Запасной способ — через pip

> Используйте этот способ только если `uv` по каким-то причинам недоступен.

Убедитесь, что Python установлен (версия 3.11+, скачать можно на [python.org](https://www.python.org/downloads) — в установщике **отметьте галочку** `Add Python to PATH`), затем создайте виртуальное окружение:

```bash
python -m venv myenv

source myenv/bin/activate   # Linux / macOS
myenv\Scripts\activate       # Windows
```

После активации установите зависимости:

```bash
pip install -r requirements.txt
# или вручную:
pip install pandas numpy matplotlib osmnx geopandas
```

#### 5.1. Запуск ячеек

- **Выполнить ячейку**: `Shift + Enter`.
