# Настройка рабочей среды

В этом разделе мы рассмотрим два способа настройки рабочей среды: какой редактор кода выбрать, как создать виртуальное окружение и как организовать хранение рабочих файлов.

Если у вас уже всё настроено, этот раздел можно смело пропустить.

> **Рекомендуемая версия Python: 3.11.14.** Именно с ней тестировались все материалы курса, установка и совместимость библиотек.

## Jupyter Notebook

**Jupyter Notebook** – это интерактивная среда для создания и обмена документами, содержащими код, текст, изображения и другие элементы. Он позволяет создавать и выполнять код в отдельных ячейках, что делает процесс работы с данными более интерактивным и удобным. Jupyter Notebook поддерживает множество языков программирования, включая Python, R, Julia и другие.

## Редакторы кода - Visual Studio Code (VS Code)

В рамках этого курса мы предлагаем работать с [Visual Studio Code (VS Code)](https://code.visualstudio.com). Это лёгкий и мощный текстовый редактор, разработанный компанией Microsoft.

### 1. Установка необходимых инструментов

#### 1.1. Установите VS Code

1. Скачайте с официального сайта: [https://code.visualstudio.com](https://code.visualstudio.com).
2. Запустите установщик и следуйте инструкциям.

#### 1.2. Установите uv

`uv` — это современный и очень быстрый менеджер пакетов и виртуальных окружений для Python. Он умеет самостоятельно скачивать и устанавливать нужные версии Python, создавать изолированные окружения и управлять зависимостями.

**macOS / Linux:**

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

**Windows (PowerShell):**

```powershell
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

После установки перезапустите терминал и проверьте:

```bash
uv --version
```

#### 1.3. Установите Python 3.11.14 через uv

Вместо того чтобы скачивать Python вручную с python.org, используем `uv` — он сам загрузит нужную версию:

```bash
uv python install 3.11.14
```

Проверьте, что версия доступна:

```bash
uv python list
```

#### 1.4. Установите расширение Jupyter в VS Code

1. Откройте VS Code.
2. Перейдите во вкладку **Расширения** (Ctrl+Shift+X).
3. Найдите **"Jupyter"** (разработчик: Microsoft) и установите.
4. Найдите **"Python"** и установите.

### 3. Виртуальное окружение с uv (рекомендуемый способ)

`uv` создаёт окружение быстро и сразу с нужной версией Python.

#### 3.1. Инициализируйте проект

Откройте терминал в VS Code (Terminal → New Terminal) и выполните:

```bash
uv init
```

Команда создаст в папке файлы `pyproject.toml` и `.python-version` — они описывают проект и фиксируют версию Python.

#### 3.2. Проверьте версию Python в окружении

```bash
uv run python --version
```

Если версия не `3.11.14`, задайте её явно:

```bash
uv python pin 3.11.14
```

После этого пересоздайте окружение:

```bash
uv sync
```

#### 3.3. Активируйте окружение вручную (при необходимости)

```bash
# Linux / macOS
source .venv/bin/activate

# Windows
.venv\Scripts\activate
```

> После активации VS Code должен автоматически подхватить окружение для ноутбука. Если нет — выберите его вручную через **Select Kernel** в правом верхнем углу редактора.

### 4. Установка библиотек

#### Рекомендуемый способ — через uv

В папке проекта уже есть файл `requirements.txt` со всеми необходимыми библиотеками. Установите их командой:

```bash
uv add -r requirements.txt
```

`uv` установит пакеты, обновит `pyproject.toml` и создаст `uv.lock` — файл, фиксирующий точные версии зависимостей.

#### Запасной способ — через pip

Если `uv` по каким-то причинам недоступен, используйте стандартный `pip`. Для этого сначала убедитесь, что Python установлен (версия 3.11+, скачать можно на [python.org](https://www.python.org/downloads) — в установщике **отметьте галочку** `Add Python to PATH`), затем создайте виртуальное окружение:

```bash
python -m venv myenv

source myenv/bin/activate   # Linux / macOS
myenv\Scripts\activate       # Windows
```

После активации установите зависимости:

```bash
pip install -r requirements.txt
# или вручную:
pip install pandas numpy matplotlib osmnx geopandas
```

### 5. Основные функции

#### 5.0. Что такое ядро (Kernel)

Ядро (kernel) — это процесс, который выполняет код из Jupyter Notebook.

Когда вы нажимаете `Shift + Enter`, код из ячейки отправляется в ядро:

- ядро выполняет код (например, на Python);
- результат возвращается обратно в ноутбук.

Если ядро не работает или зависло, код выполняться не будет.

#### 5.1. Запуск ячеек

- **Выполнить ячейку**: `Shift + Enter`.
- **Добавить ячейку**:
  - Код: нажмите `+` на панели или `Ctrl+M B`.
  - Текст (Markdown): `Ctrl+M M`.

### 6. Дополнительные рекомендации

- Горячие клавиши:
  - `Ctrl + /` — закомментировать код.
  - `Alt + Up/Down` — переместить ячейку.
- Если ядро не отвечает:
  - Перезапустите его через `Ctrl+Shift+P → Jupyter: Restart Kernel`.
